This plugin will enable Awesome Bootstrap Alert box in your Wordpress theme.

== Description ==

= Plugin Features =

* Easy to use.
* Alert Button in Editor.
* TinyMCE Button added for generating Shortcode.
* Customizable.

== Installation ==

= Installation =

1. Upload 'tcbd-alert' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

= How to use? =

1. Button: You find TCBD Alert button in Editor. Just Click what you want and fill information.Than just click OK.
2. Custom use: Just use rel attribute in anchor or span. Example code is given below.

`[tcbd-alert type="success" place="top"]Lorem ipsum dolor sit amet, consectetuer adipiscing elit.[/tcbd-alert]`
